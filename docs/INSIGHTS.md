# Insights Report

## Objective
Analyze operational KPIs for dispatch workflow performance.

## Key findings
- Lead time increases when orders remain too long in IN_PREPARATION.
- Incidents are concentrated in specific regions and channels.

## Recommendations
- Trigger alerts when status is unchanged for more than N hours.
- Improve notification routing between Dispatch and Sales.

## Limitations
- Dataset is synthetic and used for demonstration purposes.
